# Sleep-checker
# 睡眠負債チェッカー (sleep-debt-calculator)

厚生労働省の最新睡眠ガイドラインに完全準拠した、夜勤対応型睡眠負債可視化アプリケーションです。
ブログ「おしえてかめさん」のブランドイメージをベースにした、学習者向けの清潔で親しみやすいデザインを採用しています。

## 特徴
- **ホーム画面アイコン対応**: スマートフォンのホーム画面にショートカットを作成した際、指定のカスタムアイコン（`IMG_4320.png`）が表示されます。
- **夜勤・当直モード**: 夜勤前後の仮眠や、夜勤明けの変則的な睡眠パターンをバグなく正しく評価するロジックを搭載。
- **厚労省基準セルフチェック**: 睡眠不足に伴う心身の危険サインをその場でセルフ判定。

## 公開方法（GitHub Pages）
1. 本リポジトリの直下に `index.html`, `style.css`, `script.js`, `manifest.json` を配置します。
2. 同一階層にアイコン画像ファイルを **`IMG_4320.png`** という名前で保存してコミットします。
3. GitHubリポジトリの `Settings` > `Pages` から公開（デプロイ）を有効化してください。
